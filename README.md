# 💰 AI-Powered Expense Tracker

An intelligent expense tracking system that transforms raw financial data into **actionable insights and spending intelligence**.

🚀 Why This Project Exists

Most people track expenses.
Very few understand their financial behavior.

This project bridges that gap by combining:

* Data tracking
* AI-based categorization
* Insight generation

🎯 What It Does

* 📥 Add or upload expenses (CSV/manual)
* 🧠 Automatically categorize transactions
* 📊 Visualize spending patterns
* 💡 Generate smart financial insights

🛠️ Tech Stack

* **Python** — Core logic
* **Pandas** — Data processing
* **SQLite** — Local database
* **Streamlit** — UI/dashboard
* **Matplotlib / Plotly** — Visualization
* **Scikit-learn** *(optional)* — ML categorization

📊 Key Features

* Expense tracking (manual + CSV)
* Rule-based AI categorization
* Monthly & category-wise analysis
* Insight engine for savings suggestions
* Clean interactive dashboard

🧠 How It Works

1. User inputs expense data
2. Data is cleaned and structured
3. Categorization engine assigns categories
4. Analytics engine generates trends
5. Insight engine provides recommendations
